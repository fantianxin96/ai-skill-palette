# 灵感指令台版本记录

## v1.0.0｜2026-04-28

正式 1.0 版本。这个版本基于前期 `v2.1 / build 210` 的视觉与交互打磨结果，作为对外分享和 Git 存档的稳定发布点。

### 定位

- 产品名：灵感指令台
- 英文名：AI Skill Palette
- 版本名：v1.0
- Git tag：v1.0.0

### 保留的核心体验

- 使用 `⌘⇧K` 唤起全局悬浮窗口。
- 按问题场景浏览 UX / 产品设计师常用 AI prompt。
- 支持全局搜索。
- 点击卡片复制 prompt，不自动发送，避免误触。
- 左侧导航固定行高，切换更顺滑。
- 视觉收敛为柔和粉紫、暖白和中性色体系。

### 回退说明

如果后续继续大改，可以通过 Git tag `v1.0.0` 回到这个正式版本。
