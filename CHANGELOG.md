# 灵感指令台版本记录

## v1.0.1｜2026-04-28

左侧导航命中区域修复版。产品可见版本仍显示 `v1.0`，构建版本更新为 `101`。

### 修复

- 左侧导航从 `ScrollView + Button` 改为固定列表行，避免滚动手势和点击手势竞争。
- 每个导航项明确设置整行点击热区，点击空白区域也能切换。
- 左侧导航项合并为单一可访问性元素，避免图标和文字被识别成多个重复按钮。

## v1.0.0｜2026-04-28

正式 1.0 版本。这个版本基于前期 `v2.1 / build 210` 的视觉与交互打磨结果，作为对外分享和 Git 存档的稳定发布点。

### 定位

- 产品名：灵感指令台
- 英文名：AI Skill Palette
- 版本名：v1.0
- Git tag：v1.0.0

### 保留的核心体验

- 使用 `⌘⇧K` 唤起全局悬浮窗口。
- 按问题场景浏览 UX / 产品设计师常用 AI prompt。
- 支持全局搜索。
- 点击卡片复制 prompt，不自动发送，避免误触。
- 左侧导航固定行高，切换更顺滑。
- 视觉收敛为柔和粉紫、暖白和中性色体系。

### 回退说明

如果后续继续大改，可以通过 Git tag `v1.0.0` 回到这个正式版本。
