# 灵感指令台版本记录

## v1.0.6｜2026-04-28

设计系统与多页面一致性能力补充版。产品可见版本仍显示 `v1.0`，构建版本更新为 `106`。

### 新增

- 新增 Codex skill：`$i-system`，用于从满意页面提取颜色、字体、间距、组件、状态、移动端规则和文案语气，并沉淀为 `DESIGN.md`。
- 新增 Codex skill：`$i-consistency`，用于检查第二页、第三页是否与参考页面或 `DESIGN.md` 保持一致，重点覆盖字体、颜色、间距、组件、交互状态和文案语气。
- 灵感指令台新增「多页面一致性」卡片，放入「找问题」场景。
- 灵感指令台新增「设计系统沉淀」卡片，放入「删减沉淀」场景。

### 说明

- 这次不改变现有视觉、hover、导航和复制交互。
- 新增能力主要解决“第一页满意，后续页面字体、颜色、间距跑偏”的真实工作流问题。

## v1.0.5｜2026-04-28

左侧导航 row-card hover 修正版。产品可见版本仍显示 `v1.0`，构建版本更新为 `105`。

### 优化

- 左侧导航每一项都改成完整 row-card，hover 范围和选中态范围保持一致。
- 左侧导航 hover 宽度、高度、圆角与选中态一致，只是视觉强度更轻。
- 修正父级布局约束，让左侧 row 确实按完整导航卡片区域响应，而不是只跟随图标和文字内容。

## v1.0.4｜2026-04-28

Hover 层级收敛版。产品可见版本仍显示 `v1.0`，构建版本更新为 `104`。

### 优化

- 右侧 skill 卡片 hover 时只反馈整张卡片，不再改变 emoji 图标底色。
- 左侧导航 hover 时只反馈整行导航，不再改变单独图标底色。
- 复制 pill 不再因为卡片 hover 变重，避免多个元素同时抢注意力。

## v1.0.3｜2026-04-28

右侧卡片微交互增强版。产品可见版本仍显示 `v1.0`，构建版本更新为 `103`。

### 优化

- 右侧每张 skill 卡片新增 hover 反馈，鼠标经过时整张卡片轻微提亮、抬起并增强描边。
- 右侧卡片新增按下反馈，点击复制时整张卡片有克制的缩放感。
- 复制按钮和 emoji 图标随卡片 hover 状态轻微响应，让主要操作对象更有手感。
- 不新增 loading，不修改结果切换和面板开合，保持 v1.0 的安静工具感。

## v1.0.2｜2026-04-28

左侧导航微交互增强版。产品可见版本仍显示 `v1.0`，构建版本更新为 `102`。

### 优化

- 左侧导航新增 hover 反馈，鼠标经过时出现轻微底色和描边。
- 左侧导航新增按下反馈，点击瞬间有克制的缩放感。
- 选中态过渡更柔和，减少“静态切换”的平面感。

## v1.0.1｜2026-04-28

左侧导航命中区域修复版。产品可见版本仍显示 `v1.0`，构建版本更新为 `101`。

### 修复

- 左侧导航从 `ScrollView + Button` 改为固定列表行，避免滚动手势和点击手势竞争。
- 每个导航项明确设置整行点击热区，点击空白区域也能切换。
- 左侧导航项合并为单一可访问性元素，避免图标和文字被识别成多个重复按钮。

## v1.0.0｜2026-04-28

正式 1.0 版本。这个版本基于前期 `v2.1 / build 210` 的视觉与交互打磨结果，作为对外分享和 Git 存档的稳定发布点。

### 定位

- 产品名：灵感指令台
- 英文名：AI Skill Palette
- 版本名：v1.0
- Git tag：v1.0.0

### 保留的核心体验

- 使用 `⌘⇧K` 唤起全局悬浮窗口。
- 按问题场景浏览 UX / 产品设计师常用 AI prompt。
- 支持全局搜索。
- 点击卡片复制 prompt，不自动发送，避免误触。
- 左侧导航固定行高，切换更顺滑。
- 视觉收敛为柔和粉紫、暖白和中性色体系。

### 回退说明

如果后续继续大改，可以通过 Git tag `v1.0.0` 回到这个正式版本。
